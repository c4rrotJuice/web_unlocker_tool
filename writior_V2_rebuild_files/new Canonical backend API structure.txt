  title: Writior V2 API
  version: 2.0.0
  summary: Canonical backend API for the Writior V2 research-to-writing product
  description: |
    Writior V2 is a structured research workflow product built around the canonical graph:

      Sources -> Citations -> Quotes -> Notes -> Documents

    The backend is a fresh canonical rebuild. It must use:
    - bearer-token authenticated API requests
    - centralized entity serializers
    - shared ownership and capability enforcement
    - atomic relation replacement endpoints
    - secure extension-to-web handoff
    - canonical extension capture flows
    - no legacy cookie auth bridge
    - no legacy schema fallback behavior

servers:
  - url: https://app.writior.com

  - url: https://web-unlocker-tool.onrender.com
    description: Secondary deployment host, not canonical auth origin
  - url: http://localhost:5000
    description: Local development
